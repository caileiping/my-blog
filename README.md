# 我的博客

> 一个温暖文艺风格的个人博客

## 在线访问

🔗 **https://yourusername.github.io/my-blog/**

（把 `yourusername` 换成你的 GitHub 用户名）

## 本地预览

直接在浏览器中打开 `index.html` 即可

## 部署步骤

详见 [DEPLOY.md](./DEPLOY.md)

## 添加文章

编辑 `js/data.js` 文件，参考已有格式添加新文章

## 技术栈

- HTML5 + CSS3 + JavaScript
- 纯静态网站，无需服务器
- 部署在 GitHub Pages（免费）
